# The Singularity 奇点 V4.0
这是一个为 SillyTavern 设计的综合增强插件，将 记忆收藏 (Singularity) 与 聊天数据分析 (Obsession) 两大核心功能融合于一体。通过优雅的 UI 界面，帮助用户更好地管理与角色的珍贵回忆及交互数据。

✨ 主要功能
1. 🧠 记忆回廊
一键收藏：在聊天界面的每一条消息旁自动生成 👁️‍🗨️ 按钮，点击即可将该对话片段永久收录至记忆库。

记忆列表：提供沉浸式的全屏界面查看所有收藏的记忆片段，支持关键词搜索。

音频支持：支持关联音频播放，并在浏览记忆时同步播放背景音（如已配置）。

数据导出：支持将收藏的记忆一键打包导出为 ZIP 备份。

旧数据迁移：内置数据修复工具，可将旧版 Blob 格式的音频数据自动迁移为服务器文件。

2. 🕷️ 执念数据
全维统计：实时分析你与角色的交互数据，包括双方对话字数对比、回复比例。

词云分析：自动生成高频词汇云。

时间突触：

24小时活跃图：展示你们最常在哪个时间段聊天。

30天趋势图：回顾近期的互动热度。

全局搜索：支持在海量历史记录中快速搜索关键词，并收藏进记忆回廊。

3. 🎨 沉浸式 UI
无缝集成：插件菜单嵌入至 SillyTavern 的扩展栏 (Extensions) 中。

响应式设计：完美适配桌面端与移动端，全屏模态框提供最佳阅读体验。

📖 使用方法
1. 启用插件
刷新页面，打开 Extensions (扩展) 面板。

找到 👁️‍🗨️ The Singularity 菜单栏。

2. 收藏记忆 (Singularity)
在聊天框中，每一条消息的时间戳或名字旁会出现一个 👁️‍🗨️ 图标。

点击图标，提示 "记忆碎片已捕获" 即表示收藏成功。

图标变为 🧬 (紫色) 代表该条目已入库。

3. 查看与分析
点击扩展栏中的 👁️‍🗨️ The Singularity 展开控制面板：

点击 🧠 记忆回廊：

浏览、搜索、编辑或删除已收藏的记忆。

点击右上角的文件图标导出备份。

点击回收图标迁移旧版数据。

点击 🕷️ 执念数据：

查看详细的字数统计、词云和活跃度图表。

使用顶部搜索栏检索所有历史对话。

# ⚠️ 注意事项
首次使用：如果是从旧版插件升级，一定要先进入“记忆回廊”点击迁移按钮 (♻️) 以确保音频文件正确加载。
